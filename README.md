# ORZU

ORZU is an intelligent study planning platform designed to help students prepare effectively for SAT and IELTS examinations.

## Features

* Personalized study plans
* SAT and IELTS goal tracking
* Daily task management
* Progress tracking
* Study streak system
* Weekly progress analytics
* Completion statistics
* Responsive mobile-friendly design
* Local data persistence using LocalStorage

## How It Works

Users enter:

* Exam type (SAT, IELTS, or both)
* Current score/band
* Target score/band
* Daily study hours
* Exam date

ORZU then generates a personalized study plan, tracks progress, calculates study streaks, and provides analytics to help students stay consistent and achieve their goals.
